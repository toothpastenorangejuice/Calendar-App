# Calendar App
- A calendar application  built with HTML, CSS, JavaScript, and ElectronJS
- This project i learnt from Nashallery's youtube video titled How to code your first cute application from scratch (beginner friendly & aesthetic :3).

## My Contributions
- **Custom styling** -  A bit of redesign with different colours and spacing
- **Additional features** - A day function to display the day of the week. 

## What's in this repo

| File / Folder | Purpose |
| --- | --- |
| `assets/` | Images for backgrounds and containers |
| `.gitignore` | Files and folders to be ignored by Git |
| `index.html` | The main HTML layout  |
| `main.js` | Main Electron process that opens app and holds UI|
| `package-lock.json` | Auto-generated lock file for dependencies |
| `package.json` | Project Dependencies |
| `README.md` | Provides nformation about this calendar project |
| `script.js` | Frontend JavaScript controlling calendar logic |
| `styles.css` | All styles for the calendar interface |

## How to use

### 1. Clone This Repo

```
git clone https://github.com/toothpastenorangejuice/Calendar-App.git
cd calendar-app
```

### 2. Install Node.js

Make sure to install Node.js then run:

```
npm install
```

### 3. Open App

Run:

```
npm start
```
